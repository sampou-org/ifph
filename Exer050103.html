練習問題 5.1.3

0や負の数が扱えるようにするにはどうすればよいか示せ．

convertで場合分けをする．

convert               ::  Int -> String
convert n  | n <   0  =   "minus " ++ convert9 (negate n)
           | n ==  0  =   "zero"
           | n >   0  =   convert9 n
