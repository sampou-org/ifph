練習問題 1.2.1

x×y
を評価するためには，式 
x
と式 
y
を正規形に簡約してから乗算する． 
square infinity
の評価は停止するか． 





簡約は以下のように進む．


    square infinity
=   { squareの定義より }
    infinity × infinity
=   { 左のオペランドを簡約，infinityの定義より }
    (infinity + 1) × infinity
=   { 左のオペランドを簡約，infinityの定義より }
    ((infinity + 1) +1) × infinity
    ...


×

の左オペランドを簡約するプロセスが停止しない．したがって

square infinity

の評価は停止しない．
