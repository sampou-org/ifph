練習問題 1.2.2

式 
square (3+4)
に対して停止する簡約系列はいくつあるか． 





以下の3通り．ただし，グラフ簡約や並列簡約は考慮していない．


(1)


    square (3+4)
=   { 引数を簡約 }
    square 7
=   { squareの定義 }
    7 × 7
=   { ×によるかけ算 }
    49


(2)


    square (3+4)
=   { squareの定義 }
    (3+4) × (3+4)
=   { ×の左オペランドを簡約 }
    7 × (3+4)
=   { ×の右オペランドを簡約 }
    7 × 7
=   { ×によるかけ算 }
    49


(3)


    square (3+4)
=   { squareの定義 }
    (3+4) × (3+4)
=   { ×の右オペランドを簡約 }
    7 × (3+4)
=   { ×の左オペランドを簡約 }
    7 × 7
=   { ×によるかけ算 }
    49
