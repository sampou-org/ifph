練習問題 3.2.3

法則 
x↑(m+n) = (x↑m)×(x↑n)
は任意の自然数 
x
， 
m
， 
n
に対して成り立つか． 





n

上の帰納法の場合分けに

n = ⊥

の場合を追加する．




左辺
    x ↑ (m + ⊥)
=   { + の定義．場合の枯渇 }
    x ↑ ⊥
=   { ↑の定義．場合の枯渇 }
    ⊥

右辺
    x ↑ m × x ↑ ⊥
=   { ↑の定義．場合の枯渇 }
    x ↑ m × ⊥
=   { ×の定義．場合の枯渇 }
    ⊥

したがって，法則

x↑(m+n) = (x↑m)×(x↑n)

は任意の自然数

x

，

m

，

n

に対して成り立つ．
