練習問題 1.3.2

関数 
h
を等式 
h x = f (g x)
で定義したものとせよ． 
f
と 
g
がともに正格であれば， 
h
も正格であることを示せ． 





h ⊥ = ⊥

であることを以下のように示す．




    h ⊥
=   { h の定義 }
    f (g ⊥)
=   { g は正格であるから g ⊥ = ⊥ }
    f ⊥
=   { f は正格であるから f ⊥ = ⊥ }
    ⊥
