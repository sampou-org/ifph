練習問題 4.3.9

pairs n
が 
1 ≦ x, y ≦ n
を満たすすべての整数の対 
(x,y)
の重複ない列挙になるような関数 
pairs
を定義せよ． 





pairs :: Int -> [(Int,Int)]
pairs n = [(x,y) | x <- [1..n], y <- [x..n]]
