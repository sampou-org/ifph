練習問題 1.4.2

先に定義した関数 
delta
の引数をカリー化して， 
delta (a,b,c)
と書かずに， 
delta a b c
と書けるようにしたとする． このカリー化された版の型はどうなっているか． 





delta' :: (Float,Float,Float) -> Float
delta' (a,b,c) = sqrt (square b - 4 * a * c)

delta        ::  Float -> (Float -> (Float -> Float))
delta a b c  =   delta' (a,b,c)
