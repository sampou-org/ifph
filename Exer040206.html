すべての無限リスト

xs

について

reverse xs = ⊥

であることの証明




xs上の帰納法で示す．

⊥の場合

    reverse ⊥
=       { 場合の枯渇 }
    ⊥

x:xs の場合

    reverse (x:xs)
=       { reverseの定義 }
    reverse xs ++ [x]
=       { 帰納法の仮定 }
    ⊥ ++ [x]
=       { 場合の枯渇 }
    ⊥

すべての無限リスト

xs

について

init xs = ⊥


であることの証明．


xs上の帰納法で示す．

⊥の場合

    init ⊥
=       { initの定義 }
    (reverse . tail . reverse) ⊥
=       { (.)の定義 }
    reverse (tail (reverse ⊥))
=       { 直前の証明 }
    reverse (tail ⊥)
=       { 場合の枯渇 }
    reverse ⊥
=       { 直前の証明 }
    ⊥

x:xs の場合

    init (x:xs)
=       { initの定義 }
    (reverse . tail . reverse) (x:xs)
=       { (.)の定義 }
    reverse (tail (reverse (x:xs)))
=       { 直前の証明 }
    reverse (tail ⊥)
=       { 場合の枯渇 }
    reverse ⊥
=       { 直前の証明 }
    ⊥
