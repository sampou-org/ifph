練習問題 2.1.2

論理において⇒で表される含意は，x⇒yが 
False
になるのはxが真でかつyが偽の場合に限ると定義されている． 含意を 
Bool
上の演算として形式的な定義を与えよ． 




⇒は

=>>

と書くことにする．




infixr 1 =>>

(=>>) :: Bool -> Bool -> Bool
p =>> q = not p \/ q
