練習問題 1.5.2

整数の絶対値を返す関数 
abs::Integer -> Integer
を定義せよ． 





abs

は

Num

クラスのメソッドとして定義済みなので

abs'

として定義する．




abs' :: Integer -> Integer
abs' x  = if x >= 0 then x else -x
