練習問題 3.2.4

+
が結合的であること，すなわち，任意の自然数 
m
， 
n
， 
p
に対して 
(m+n)+p = m+(n+p)
であることを証明せよ． 





p

上の帰納法




⊥の場合
左辺
   (m + n) + ⊥
=  { + の定義．場合の枯渇 }
   ⊥
右辺
   m + (n + ⊥)
=  { + の定義．場合の枯渇 }
   m + ⊥
=  { + の定義．場合の枯渇 }
   ⊥

Zero の場合
左辺
   (m + n) + Zero
=  { + の定義 }
   m + n
右辺
   m + (n + Zero)
=  { + の定義 }
   m + n

Succ p の場合
   (m + n) + Succ p
=  { + の定義 }
   Succ ((m + n) + p)
=  { 帰納法の仮定 }
   Succ (m + (n + p))
=  { + の定義 }
   m + Succ (n + p)
=  { + の定義 }
   m + (n + Succ p)
