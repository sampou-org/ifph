練習問題 2.5.1.

変換元の型が 
Either Bool Char
で， 
Left ⊥
， 
Right ⊥
， 
⊥
の3種類の引数に対して別々の振る舞いになるような関数を定義せよ． 





isLeft            ::  Either Bool Char -> Bool
isLeft (Left _)   =   True
isLeft (Right _)  =   False
